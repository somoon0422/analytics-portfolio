01
Problem

데이터는 있는데
분석이 어려운 현실

비즈니스 인사이트가 필요한 실무자들이 SQL을 몰라 매번 개발팀에 데이터를 요청해야 했습니다. 단순한 조회 하나에도 며칠씩 걸리는 비효율이 발생했습니다.

긴 대기 시간

단순 데이터 조회에도 개발팀 요청 후 2-3일 소요

SQL 진입 장벽

비개발 직군의 복잡한 쿼리 작성 어려움

반복 작업

유사한 분석을 매번 처음부터 다시 요청

데이터 요청 병목
개발팀 리소스의 30%가 단순 데이터 조회에 소모
